# Organize Choir

## Comandos

**Criar pastas dos naipes e dos cantores**
```powershell
    npm run dev mkdir .\cantores.json
```
**Renomear arquivos dos cantores**
```powershell
    npm run dev rename
```
**Calcular tamanho de todos os arquivos dos cantores**
```powershell
    npm run dev size
```
